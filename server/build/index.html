Automated Term Extraction Demonstrater

Web app to demonstrate term extraction algorithms, using PyATE as backend (https://github.com/kevinlu1248/pyate). When text is entered into the input, the list of predicted terms as well as their "termhoods" will be listed. The termhood is a value correlated with the probability of each phrase being a term, that is, it is a value denoting how much the phrase looks like a term.

ComboBasic is an extension of the Basic algorithm, designed by Astrakhantsev (2016), which calculates the termhood of a candidate term t by a weighted average of the number of times t contains another candidate, the number of times another term contains t, and the length of t in characters times the log of the frequency of t.
